Ovearall objective of this project is to create two API's to handle
- Menu creation
- Placing Orders

As part of Menu Creation:
- Menu_ID
- Name of the Menu
- Spicy - yes or no
- Veg or Non Veg
- Type of Menu: Starter/Main Course/Deserts/
- Price of the menu will be listed


As part of Placing Orders:
- Table No
- Name of the Menu
- Menu Id
- Quantitiy
- Price
